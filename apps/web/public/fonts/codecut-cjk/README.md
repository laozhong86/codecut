# CodeCut CJK Fonts

These fonts are bundled for local CodeCut preview and node export parity.

Only fonts with redistributable license evidence are included here. Commercial
reference fonts from source videos must not be copied into this directory unless
their license explicitly allows redistribution with CodeCut.
