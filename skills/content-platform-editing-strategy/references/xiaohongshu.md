# Xiaohongshu

Use this as a China-market dated hypothesis. Run a current-source check for latest Xiaohongshu creator, search, commerce, or sponsored-content guidance. Do not claim secret ranking weights.

## Editing bias

Xiaohongshu editing should usually optimize for trust, search usefulness, save value, and lifestyle fit more than hard-sell urgency. A video should feel like a useful note with visible evidence, not only an ad cutdown.

Good structures:

- User problem -> personal context -> proof or comparison -> practical takeaway.
- Before state -> product/process -> result -> save-worthy checklist.
- Search query answer -> 3 evidence-backed points -> soft CTA.

## Content strategy

- `audienceIntent`: compare, decide, save for later, search a solution, or validate a purchase.
- `contentAngle`: note-like review, checklist, before/after, mistake correction, personal experience, or lifestyle proof.
- `scriptPromise`: a useful answer the viewer can remember or save.
- `proofAsset`: cover/title match, texture/detail shot, comparison, usage step, limitation, result, or decision criteria.
- `interactionDesign`: save, ask for fit/context, search follow-up, consult, or soft purchase cue.
- `editingImplication`: preserve useful details, keep caption language searchable, and avoid sacrificing trust for speed.

## Evidence to inspect

- Cover/title promise and whether the first frame supports it.
- Search keywords in title, opening line, overlays, and captions.
- Save-worthy elements: checklist, comparison, measurements, steps, or decision criteria.
- Trust cues: creator use, real scene, texture detail, before/after, honest limitation.
- Whether the tone matches lifestyle, review, tutorial, or recommendation intent.

## Avoid

- Hard TikTok-style CTA stacking when the user expects a note-like recommendation.
- Over-polished commercial visuals that weaken authenticity.
- Missing cover/title strategy for search discovery.
- Unsupported "must buy" or medical/financial/beauty efficacy claims.
- Replacing useful detail with only fast pacing.
